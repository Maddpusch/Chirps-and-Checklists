Data Input

R Packages

Add the packages we need into R. The Vegan package is for calculating Hill diversity indices, the rarefied richness, and rarefaction curves. The SpadeR package was used to calculate the Chao1 estimator, and the dplyr package was used to increase functionality.

Input Data from csv File

The data from eBird was filtered to only include data from the Whitemud Park hotspot in Edmonton, Alberta from the months that standardized data was also extracted for. The months are as follows;

2016-2017: December, January, February

2017-2018: December, January, February

2018-2019: January

2020-2021: January, February

2021-2022: December, January

2022-2023: January, February

For both the citizen and standardized data, the observations with unspecified species as well as observations lacking a count of the individuals observed were removed.

This filtered data was then formatted as a csv and inputted into R.

Whitemud5.dat<-read.csv("Whitemud Data EBird_R.5.9.csv")

Hill Indices

We separated the data based on sampling method (standardized vs citizen), as well as year. Did this by filtering the “SumCode”, which is a shorthand for each data point that gives the season from the last two digits of the years it is from (ex: “16-17” would be the 2016-2017 season) and the type of data it is (“ST” for standardized sampling, and “CI” for citizen sampling).

Then we created a vector of abundances so Hill numbers could be calculated, and then calculated the Hill numbers and recorded them.

We completed this process individually for citizen and standardized science from each season.

Rarefaction Curves

Rarefaction by Season

To create a visually smooth curve, we ran the rarefaction curves with a few different starting numbers and increasing values. These never changed the results, but only affected how smooth the rarefaction curve appeared on the graph. In the end, 10 was chosen as the starting number and increasing value. The end number was set as the total number of individuals from the data set being calculated.

#16-17:
ST16.17.nsample10<-seq(10,907,by=10)
CI16.17.nsample10<-seq(10,5451,by=10)
rich.ST16.17_10<-rarefy(ST16.17.SpComp,ST16.17.nsample10)
rich.CI16.17_10<-rarefy(CI16.17.SpComp,CI16.17.nsample10)

plot(ST16.17.nsample10, rich.ST16.17_10,type="l", xlab="Number of individuals", ylab="Rarefied bird species richness", lwd=4, col="slateblue", ylim=c(0,50), xlim=c(0,5460))
points(CI16.17.nsample10,rich.CI16.17_10,type="l", col="darkolivegreen3", lwd=4,lty=3)

Asymptotic Richness Estimator (Chao1)

Chao1 was used to estimate total species richness by using a preset equation of the singletons and doubletons of species abundance found in the data. The Output table was limited after the first data set to only include the output we are looking for.

#estimated richness = sample richness plus (singletons-(singletons-1))/(2 times(doubletons plus 1))
